MService Admin UI

Gaterace Microservices Suite

MService is a microservice for authentication and authorization in support of this and other microservices. It is written in the language Go, and uses gRPC to define and implement it's application programming interface (API). A successful invocation of the login api returns a JSON Web Token (JWT) that can be used with this and other microservices to gain access to methods in those microservices.

If the logged-in user has sufficient admin-level authority, she will be able to manage Account, Claims, Users and Roles data using this application.
