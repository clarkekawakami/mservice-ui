{{ if eq .active "mservice"}}

The MService microservice provides authentication and authority credential management services.

If the logged in user is in the master account and is assigned the admin role, then she has R/W access to:

  • All accounts
  • All claims and claim names
  • All roles for all accounts
  • All users for all accounts

If the logged in user is not in the master account but is assigned the admin role, then she has R/W access to:

  • Her account
  • All roles for her account
  • All users for her account

If the logged in user is not assigned the admin role, then she has R/W access to:

  • Her password
  • Her user record
